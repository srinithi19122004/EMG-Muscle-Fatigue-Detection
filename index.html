About this system

This system detects muscle fatigue using surface EMG signals captured via Myoware 2.0 sensor and skin electrodes. Features extracted: RMS, MDF, ZCR, STD, MAV. Classification uses thresholds derived from published EMG research (De Luca 1997, Merletti 2004). An Edge Impulse neural network model provides secondary classification for neuromuscular disorder pattern detection.
